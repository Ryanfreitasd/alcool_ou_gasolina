# Álcool ou Gasolina

Projeto que calcula qual combustível é mais vantajoso R$

## Features
<ul>
  <li>Insere valores de combustível álcool e gasolina</li>
  <li>Calcula qual combustível é mais vantajoso baseado no percentual de 70%</li>
</ul>
